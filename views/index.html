Free Code Camp Weather API Pass-through

USAGE:
Use this endpoint: https://fcc-weather-api.glitch.me/ to get the current weather at a location.
To prevent abuses, this server accepts GET requests only, and serves only the route /api/current?lon=:longitude&lat=:latitude. Image links that match the weather description are included in the JSON under weather[0].icon. This should return all the information you'll need to complete the user stories.

EXAMPLE:
Request: https://fcc-weather-api.glitch.me/api/current?lat=35&lon=139
Response: { "coord":{ "lon":139, "lat":35 }, "weather":[ { "id":500, "main":"Rain", "description":"light rain", "icon":"https://cdn.glitch.com/6e8889e5-7a72-48f0-a061-863548450de5%2F10n.png?1499366021399" } ], "base":"stations", "main":{ "temp":22.59, "pressure":1027.45, "humidity":100, "temp_min":22.59, "temp_max":22.59, "sea_level":1027.47, "grnd_level":1027.45 }, "wind":{ "speed":8.12, "deg":246.503 }, "rain":{ "3h":0.45 }, "clouds":{ "all":92 }, "dt":1499521932, "sys":{ "message":0.0034, "country":"JP", "sunrise":1499451436, "sunset":1499503246 }, "id":0, "name":"Shuzenji", "cod":200 }

TIPS:
Most of the information you'll need should be self-explanatory, but there are a few things we'd like to point out:

  • Inside the main field, there is a parameter called temp_min and a parameter called temp_max. These are not a daily min/max as you might expect - they are the current upper and lower bound for the main temperature. Think of these as a possible temperature range, showing the min and max value that the main temperature could be. They will often be the same value as main.temp.
  • All responses from this api are returned in metric values. It is up to you to figure out how to convert them when it's necessary.
  • The weather field is an array. Weather[0] will show you information on the primary weather conditions for the area. It is possible to have multiple conditions occuring in an area. The weather.main parameter is the overall description of the current condition: weather.description is a more detailed variant of the main condition.
This API pulls it's information from the Open Weather Map API. Should you need more detailed information on the fields returned from this call or the weather descriptions, use the related links below.
https://openweathermap.org/current
https://openweathermap.org/weather-conditions
